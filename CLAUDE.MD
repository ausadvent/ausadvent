# Project: AUSADVENT CARE Digital Engine

## Brand Identity
- **Primary:** #2563EB (Blue 600) / **Secondary:** #F59E0B (Amber 500)
- **Voice:** Professional, calming, NDIS-compliant.

## Development Standards
- **Routing:** Next.js App Router (src/app).
- **Styling:** Tailwind CSS with 'brand' prefix tokens.
- **Components:** Functional components with TypeScript interfaces.
- **Safety:** Verify environment variables for Contentful are present before client initialization.

## Essential Commands
- **Install:** `npm install`
- **Dev:** `npm run dev`
- **Lint:** `npm run lint` (Must pass zero warnings before commit).
- **Automation:** `n8n start` (Local workflow engine).

## Workflow (Vibe Coding)
- Work on `feature/` branches branching from `dev`.
- Prefix commits with `feat:`, `fix:`, or `chore:`.
- Perform a `@Codebase` scan before refactoring existing architecture.
